Problem statement: Take a name as input and print a greeting.
Input format:
    One line containing a string
Output format:
    One line containing the greeting
Sample test cases (examples):
Test case 1:
    Sample input:
        Ravi
    Expected output:
        Welcome Ravi
Test case 2:
    Sample input:
        Sailesh
    Expected output:
        Welcome Sailesh

Storage
    - Information is stored using variables
    - In Javascript, we declare variables using `let`
    - e.g.
        let name = "Ravi";
        let name = readLine();
        let x = "Sailesh";
        let userName = "Samir";

Summary
